OPENCLAW REPOSITORY DOCTRINE
	1.	This repository has a single defined purpose in the OpenClaw ecosystem.
	2.	Platform infrastructure lives in openclaw-ops, not in product or workspace repos.
	3.	Public tools with independent licensing live in their own repositories (e.g. octriageunit).
	4.	Product surfaces (marketing, docs, support) live in acme-site and acme-support.
	5.	Personal environments, agent memory, and experiments live in homarus-eximius.
	6.	If a file's purpose is unclear, stop and consult OPENCLAW_RUNTIME_SOURCE_MAP.md before editing.
    7.	Never duplicate operational code across repositories - move it to the canonical repo instead.
